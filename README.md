# 项目名：fish-boat-travel
# 网站名：渔之旅

使用了ts+组合式VUE（有些是JS），支持深色模式，支持响应式设计，组件均来自与Element-Plus\
使用了来自vue-admin-box，Vue-One piece-Admin 的部分组件\
This template should help get you started developing with Vue 3 in Vite.
